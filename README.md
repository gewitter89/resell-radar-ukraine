# Resell Radar Ukraine 🇺🇦

**Resell Radar Ukraine** — это интеллектуальный Telegram-бот для мониторинга свежих объявлений на OLX Украина и поиска выгодных товаров для перепродажи (флиппинга). Бот не просто собирает ссылки, а оценивает финансовую выгоду и риски каждой сделки, помогая ответить на главный вопрос: **«Стоит ли покупать этот товар прямо сейчас?»**

---

## 1. Возможности системы
* **Умный мониторинг**: Сканирование OLX с ротацией User-Agent, задержками и экспоненциальным backoff для защиты от банов.
* **Оценка сделки (Deal Score)**: Вычисление выгоды от 0 до 100 на основе сравнения цены с рыночной медианой, репутации объявления и соответствия «зеленому» порогу.
* **Анализ рисков (Risk Score)**: Выявление опасных факторов (проблемы с iCloud, отсутствие фото/описания, подозрительно низкая цена, требования предоплаты).
* **Финансовый трекер**: Логирование покупок и продаж, подсчет чистой прибыли и ROI (Return on Investment).
* **Система обратной связи (Обучение)**: Динамическое изменение порогов чувствительности в зависимости от реакций пользователя (кнопки `Купил`, `Интересно`, `Мусор`).
* **Защита от спама**: Лимит на отправку (не более 3 объявлений за цикл), 30-минутный cooldown на категорию, обходящийся только в случае «супер-сделок» (скидка >40%, риск <35%).

## 2. Установка и запуск

### Шаг 1: Клонирование и подготовка окружения
Убедитесь, что у вас установлен Python 3.11+.

```bash
# Перейдите в директорию проекта
cd resell_radar

# Создайте виртуальное окружение
python -m venv venv

# Активируйте виртуальное окружение
# Для Windows (PowerShell):
venv\Scripts\Activate.ps1
# Для Linux/macOS:
source venv/bin/activate

# Установите зависимости
pip install -r requirements.txt
```

### Шаг 2: Создание Telegram-бота
1. Откройте Telegram и найдите официального бота [@BotFather](https://t.me/BotFather).
2. Отправьте команду `/newbot` и следуйте инструкциям (задайте имя и юзернейм бота).
3. Скопируйте полученный **Token** (например, `123456789:ABCdefGhIJK...`).

### Шаг 3: Получение вашего `chat_id`
1. Запустите созданного бота в Telegram нажатием кнопки **/start** или отправьте команду `/start`.
2. Если вы еще не прописали `TELEGRAM_CHAT_ID` в файле `.env`, бот автоматически отклонит доступ и выведет сообщение с вашим **Chat ID** (например, `987654321`). Скопируйте это число.

### Шаг 4: Настройка переменных окружения
Создайте файл `.env` в корне проекта (или скопируйте `.env.example`):

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGhIJK...  # Токен вашего бота
TELEGRAM_CHAT_ID=987654321                  # Ваш Chat ID

CHECK_INTERVAL_MIN_SECONDS=120               # Минимальный интервал проверки (сек)
CHECK_INTERVAL_MAX_SECONDS=300               # Максимальный интервал проверки (сек)

DEAL_SCORE_THRESHOLD=75                      # Минимальный Deal Score для отправки
RISK_SCORE_MAX=45                            # Максимальный Risk Score для отправки
DEFAULT_MIN_PROFIT=1000                      # Минимальная прибыль по умолчанию (грн)

DATABASE_URL=sqlite:///resell_radar.db        # URL базы данных SQLite
```

### Шаг 5: Запуск бота
Запустите главный файл проекта:

```bash
python main.py
```
При первом запуске бот автоматически создаст локальную базу данных `resell_radar.db` и начнет фоновый мониторинг объявлений по списку из `app/data/watchlist.json`.

## 3. Управление через Telegram
Бот реагирует на следующие команды:
* `/start` — Запуск бота и приветствие.
* `/help` — Вывод справочной информации о командах.
* `/watchlist` — Просмотр списка отслеживаемых товаров и их ценовых параметров.
* `/stats` — Общая статистика (всего найдено, отправлено, куплено, мусор).
* `/profit` — Финансовый отчет (общая прибыль, средний ROI %, лучший товар и категория).
* `/top` — Список самых прибыльных категорий.
* `/pause` — Временная приостановка сканирования OLX.
* `/resume` — Возобновление сканирования.

## 4. Редактирование списка watchlist.json
Файл находится по пути `app/data/watchlist.json`. Каждый отслеживаемый элемент имеет следующую структуру:

```json
{
  "id": "iphone_12",                                                                 // Уникальный ID товара
  "name": "iPhone 12",                                                               // Понятное название для отчетов
  "category": "phones",                                                              // Категория товара
  "search_url": "https://www.olx.ua/uk/list/q-iphone-12/?search%5Border%5D=created_at:desc", // Ссылка на поиск OLX (сортировка по дате)
  "keywords": ["iphone 12", "айфон 12"],                                             // Ключевые слова для фильтрации в заголовках
  "max_green_price": 9500,                                                           // "Зеленая" цена сделки (покупка без лишних раздумий)
  "normal_price_range": [11000, 14500],                                              // Стандартный диапазон рыночных цен перепродажи
  "min_profit": 2000,                                                                // Минимальная желаемая чистая прибыль
  "bad_words": ["icloud", "копия", "реплика", "не включается", "на запчасти"]        // Запрещенные слова для этого товара
}
```

## 5. Как работают алгоритмы оценки

### Deal Score (Выгода): от 0 до 100
Оценивает привлекательность цены и оформления объявления:
* **Цена ниже рынка**: Сравнивается с медианой прошлых объявлений (или с `normal_price_range`). Если скидка `>= 30%` от медианы — начисляется **+35 баллов**; если `>= 20%` — **+25 баллов**; если `>= 10%` — **+15 баллов**.
* **Свежесть объявления**: Если опубликовано менее 10 минут назад — **+20 баллов**; до 60 минут — **+10 баллов**.
* **Полнота описания**: Текст описания длиннее 100 символов — **+10 баллов**.
* **Наличие фотографий**: В объявлении есть изображения — **+10 баллов**.
* **Ключевые слова в названии**: Совпадение бренда/модели — **+15 баллов**.
* **Зеленая зона**: Цена ниже порога `max_green_price` — **+10 баллов**.

### Risk Score (Опасность сделки): от 0 до 100
Определяет вероятность обмана или скрытого дефекта:
* **Критические слова** (`icloud`, `не включается`, `на запчасти` и т.д.): **+35 баллов** за каждое совпадение.
* **Стандартные стоп-слова** (`копия`, `реплика`, `разбит`, `после воды`): **+15 баллов** за каждое совпадение.
* **Аномально низкая цена**: Если цена составляет менее 40% от медианы рынка — **+20 баллов** (признак мошенничества или блокировки).
* **Скудное описание**: Менее 20 символов текста — **+10 баллов**.
* **Нет фотографий**: **+15 баллов**.

## 6. Auto Shopper (Grocery Price Comparison)

**Auto Shopper** — автоматическая система сравнения цен на продукты в киевских магазинах с доставкой Новой Почтой.

### Возможности
- **Zakaz.ua REST API** — реальные скидки Novus/Auchan/Motor без скрейпинга
- **3x/день авто-раннер** — 9:00, 13:00, 18:00
- **Telegram отчёты** — кликабельные ссылки на товары
- **Nova Poshta доставка** — расчёт стоимости на ваш адрес
- **Price tracking** — SQLite история цен + алерты на падение

### Запуск
```bash
# Одноразовый тест
python auto_shopper.py --test

# Фоновый режим (3x/день)
python auto_shopper.py --loop

# Или через PowerShell
.\run_shopper.ps1
```

### Конфигурация
Файл `shopping_list.json`:
- `address` — адрес доставки
- `telegram_bot_token` — токен бота
- `telegram_chat_id` — ваш chat_id
- `categories` — список продуктов по категориям

- **Groq** — Llama 3.3 70B (быстрый)
- **GitHub Models** — GPT-4.1, GPT-4o, o4-mini
- **OpenRouter** — бесплатные модели
- **Gemini** — Google AI Studio
